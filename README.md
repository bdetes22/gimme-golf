# Gimme Golf

Golf simulator rental and installation website for Gimme Golf, serving Utah with locations in Kaysville and Clearfield.

## Tech Stack

- **Framework:** Next.js 14 (App Router)
- **Language:** TypeScript
- **Styling:** Tailwind CSS

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

## Pages

- `/` — Home
- `/locations` — Locations overview
- `/locations/kaysville` — Kaysville location
- `/locations/clearfield` — Clearfield location
- `/book` — Book a simulator
- `/shop` — Shop
- `/build` — Quote builder
- `/contact` — Contact
